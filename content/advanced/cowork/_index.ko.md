#### Terraform with Github
IaC는 Infrastructure as Code 코드로써의 인프라를 이야기합니다.
Terraform 은 IaC 도구이며, 즉 테라폼은 코드를 수행합니다.
그렇기 때문에 반드시 코드로써의 관리가 필요하며, 팀원간의 코드리뷰가 필요합니다.
